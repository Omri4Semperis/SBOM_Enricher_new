## Top 3 (best effort / value)

### 1. Deterministic guard: a GT URL that isn't a fetchable license file can't fail us

**~64 rows.** The GT is often an HTML landing page (`pkg.go.dev`,
`pkgs.alpinelinux.org`, `nuget.org/.../License`). We already downloaded a real
license file; theirs won't download at all. Penalizing that is indefensible
whether or not GT is "right" as a string.

**Change:** when GT URL is not a fetchable license file (HTML content-type /
known landing-page host), do **not** grade URL as Mismatch. Treat as
unscoreable / GT-not-a-file (or compare our file against a canonical SPDX text).

**Why first:** cheapest, most certain win. No judgment call — verified those
URLs return `text/html`. Owner: comparison / grading logic (`equality.py`), not
the agent.

### 2. Split the equality verdict: TRUE / FALSE-we're-wrong / FALSE-GT-suspect

Today the judge only answers "equal?" — yes/no. It can't say "these differ, and
the evidence backs *our* side." That one label quietly holds three situations:
we're wrong, GT's wrong, both are defensible.

**Change:** third verdict when our answer is independently grounded (our license
file is real and on-topic; copyright holder matches repo/package owner). Convert
a pile of "misses" into a reviewable **GT-suspect** pile. Also tighten
over-strict FALSE on years and "X" vs "X and Contributors" as a side effect.

**Why second:** operationalizes the "human GT may be wrong" premise. Trades a
clean number for a truer, messier one that needs occasional human eyes — a
design bet, not a fact-grade fix.

### 3. Fix our two genuine errors — so "GT was wrong" stays credible

You can't claim GT was wrong when we returned *nothing*.

1. **Empty NuGet inferences (~70 rows)** — Claude returns no URL; only npm has
   a download fallback. Deterministic NuGet nuspec fallback via
   `api.nuget.org/v3-flatcontainer/{id}/{ver}/{id}.nuspec` (SPDX + repo LICENSE)
   — matches BACKLOG Lever #2.
2. **OpenTelemetry-Go copyright mis-extraction (13 rows)** — agent said
   "Copyright 2009 The Go Authors" instead of "The OpenTelemetry Authors"
   (Apache LICENSE with no holder line; extractor latched onto an unrelated
   notice).

**Why third:** cleans the signal. After this, remaining disagreements are
genuinely GT-suspect, not us hiding behind the excuse.
